# Features

## Core Features

### Timestamp Recording

- Keyboard shortcut recording (`Alt+Shift+C`)
- One-click button recording
- Works on live streams and VODs
- Optional memo support

### Record Management

- Side Panel interface
- Records grouped by streamer
- Edit and delete records
- VOD linking status display
- Completion tracking

### Clip Creation

- One-click navigation to Twitch clip creator
- Automatic position calculation
- Mark clips as completed

## Smart Features

### Automatic VOD Linking

When you record a moment during a live stream:

1. The record is saved with the broadcast ID
2. After the stream ends, visit the VOD page
3. The extension automatically matches and links records

### Floating Widget

- Draggable position (persisted)
- Auto-hide after 3 seconds
- Shows pending record count
- Click to open Side Panel

### Data Management

- All data stored locally (Chrome storage)
- No external servers
- Automatic cleanup after 60 days

## Privacy & Security

- No authentication required for basic features
- Optional Twitch connection for enhanced features
- All data stays on your device
- No analytics or tracking
