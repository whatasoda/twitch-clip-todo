# Privacy Policy

_Last updated: 2025-01-25_

## Overview

Twitch Clip Todo is committed to protecting your privacy. This extension does not collect, transmit, or store any personal data on external servers.

## Data Stored Locally

All data is stored on your device using Chrome's built-in storage APIs:

### Record Data

- Timestamps (in seconds)
- Memos (optional user notes)
- Streamer names
- VOD IDs (for linked recordings)
- Creation timestamps

### Optional Twitch Data

If you connect your Twitch account:

- Cached streamer information
- Cached VOD metadata
- Authentication tokens (stored securely)

## Permissions

| Permission  | Purpose                        |
| ----------- | ------------------------------ |
| `tabs`      | Detect current Twitch page     |
| `storage`   | Store records locally          |
| `activeTab` | Interact with Twitch player    |
| `alarms`    | Schedule automatic cleanup     |

## Third-Party Services

- **Twitch API** (optional): Used for enhanced features when authenticated
- No analytics or tracking services

## Data Deletion

To delete all data:

1. Right-click the extension icon
2. Select "Remove from Chrome"

Or clear extension data in Chrome settings.

## Contact

For questions or concerns, please open an issue on [GitHub](https://github.com/whatasoda/twitch-clip-todo/issues).
